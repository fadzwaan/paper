```bash
[root@hx8 ~]# docker run ubuntu
[root@hx8 ~]# docker ps -a
```

remember based on ps -a , you free to put any name at --name -it with IMAGE and include command

if you encounter this. just stop all related container
```bash
[root@hx8 ~]# docker rmi ubuntu
Error response from daemon: conflict: unable to remove repository reference "ubuntu" (must force) - container 4b1e76bb1bea is using its referenced image bf3dc08bfed0
```

or can use 
```
[root@hx8 ~]# docker rmi -f ubuntu
Untagged: ubuntu:latest
Deleted: sha256:bf3dc08bfed031182827888bb15977e316ad797ee2ccb63b4c7a57fdfe7eb31d
```
